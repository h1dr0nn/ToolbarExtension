# Toolbar Extension

Extend the Unity Toolbar with your own GUI code.

Based on [marijnz/unity-toolbar-extender](https://github.com/marijnz/unity-toolbar-extender).

## Features

- **Scene Switcher**: Dropdown menu to quickly switch between scenes
- **Play From Loading**: Start playmode from a loading scene, then return to your current scene
- **Clear PlayerPrefs**: One-click button to clear all PlayerPrefs
- **Recompile Scripts**: Force Unity to recompile all scripts
- **TimeScale Slider**: Adjust game speed in playmode

## Configuration

Some features require configuration. See [CONFIGURATION.md](CONFIGURATION.md) for details.

## Installation

### Via Git URL (Recommended)

1. Open Unity Package Manager (Window → Package Manager)
2. Click the `+` button → "Add package from git URL"
3. Enter: `https://github.com/h1dr0nn/ToolbarExtension.git#main`

### Manual

1. Copy the `ToolbarExtension` folder into your Unity project's `Packages` directory
2. Or add via Package Manager → "Add package from disk" → select `package.json`
